# node-badge-generator

![github](https://badge.gnlc.me/generate/github/iamgnlc/deeppink)

## Run in dev

```sh
$ yarn local
```

## Run in prod

```sh
$ yarn start
```

## How to use

E.g.: http://localhost:3001/generate/version/1.0.0/green

## Deployment

Install now cli:

```sh
$ npm i -g now@latest
```
